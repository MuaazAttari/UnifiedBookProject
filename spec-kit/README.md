# Unified Physical AI & Humanoid Robotics Learning Book

This project implements a comprehensive textbook generation and RAG (Retrieval-Augmented Generation) system for learning about Physical AI and Humanoid Robotics.

## Features

- Interactive textbook with Docusaurus frontend
- RAG-powered chatbot for answering questions from the book
- User authentication and personalization
- Urdu translation capabilities
- GitHub Pages deployment

## Tech Stack

- **Frontend**: Docusaurus
- **Backend**: FastAPI
- **Database**: Neon Postgres
- **Vector DB**: Qdrant Cloud
- **Authentication**: Better-Auth
- **AI Processing**: OpenAI API

## Getting Started

1. Clone the repository
2. Install dependencies
3. Set up environment variables
4. Run the development servers
